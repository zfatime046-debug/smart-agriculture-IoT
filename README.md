# Smart Agriculture IoT 🌱

Django-based IoT Smart Greenhouse Management System

## Features
- Sensor monitoring
- Automation rules engine
- Actionneur control
- Multi-greenhouse management
- Role-based access

## Technologies
- Django
- Python
- Mysql Workbench
